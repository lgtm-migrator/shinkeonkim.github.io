## Target #6 - Missing Slice

<img src="/assets/img/posts/2021-10-10/css-battle-06/1.png" alt="문제" class="w-50">

이것도 특이한 아이디어 없이, 그냥 구획을 4개로 나누고 각각 border-radius를 따로 적용했다.

```html
<div>
  <i></i>
  <i></i>
  <i></i>
</div>

<style>  
  body {
    background: #E3516E;
  	margin: 0;
  }
  div {
    display: flex;
    flex-wrap: wrap;
    width: 200px;
    position: absolute;
    transform: translate(-50%, -50%);
    left: 50%;
   	top: 50%;
  }
  i {
    flex-basis: 50%;
    height: 100px;
    background: #111;
  }
  
  i:nth-child(1) {
    background: #51B5A9;
    border-radius: 100% 0 0;
  }
  
  i:nth-child(2) {
    background: #FADE8B;
    border-radius: 0 100% 0 0;
  }
  
  i:nth-child(3) {
    background: #F7F3D7;
    border-radius: 0  0 0 100%;
  }
</style>

```

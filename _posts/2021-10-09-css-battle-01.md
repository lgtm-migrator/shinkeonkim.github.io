## Target #1

![문제](/assets/img/posts/2021-10-09/css-battle-01/3.png)

body에 있는 기본 margin을 0으로 만들면 쉽게 풀린다. 

도형의 px 등의 수치는 직접 알아내야 하는 것 같다.
Output을 슬라이딩하면 쉽게 정답과 비교할 수 있는 기능이 있어서 그렇게 크게 문제가 되는 것 같지는 않다 :]

```html
<div></div>
<style>
  body {
    margin: 0;
    background: #5d3a3a;
  }
  div {
    width: 200px;
    height: 200px;
    background: #b5e0ba;
  }
</style>
```
